Перед использвоанием данного приложения его необходимо скопировать на устройство
Затем требуется установить PostgreSQL
В случае Windows требется скачать его с официального сайта (https://www.postgresql.org/download/windows/)
В случае Linux - установить командой apt install postgresql
Далее необходимо зарегистрировать там пользователя:

username: postgres
name: postgres
password: 123456
port: 5432

Данное приложение запускается из командной строки
В командной строке нужно ввести 2 команды:

1) cd {директория установки программы}/EmphaSoft 
2) python manage.py runserver

Программа запустится и будет доступна по адресу http://127.0.0.1:8000/

Для доступа к панели администратора откройте страницу http://127.0.0.1:8000/admin/ и введите следующие данные:

логин: peromer
пароль: peromer1812

Просмотр своих броней пользователем осуществляется в профиле
